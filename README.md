# 📦 Controle Operacional Logístico

Demonstração do projeto

Sistema web desenvolvido para controle operacional logístico, permitindo registrar, editar, excluir e exportar ocorrências de transporte de forma prática e organizada.

O projeto foi desenvolvido para praticar:

- manipulação de DOM com JavaScript
- armazenamento local com localStorage
- exportação de planilhas XLS
- organização de layout administrativo
- estruturação profissional de HTML, CSS e JS
- interface moderna com efeito glassmorphism
- componentização e organização de código

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- SheetJS (XLSX)
- Bootstrap Icons
- Google Fonts (Poppins)
- VS Code
- Git & GitHub

---

# 🎨 Funcionalidades

✅ Cadastro de ocorrências logísticas  
✅ Edição de ocorrências  
✅ Exclusão de registros  
✅ Exportação automática para Excel (.XLSX)  
✅ Armazenamento local com localStorage  
✅ Campo de pesquisa  
✅ Interface moderna com efeito blur/glassmorphism  
✅ Dashboard operacional  
✅ Sistema de status personalizados  
✅ Navegação entre páginas  
✅ Botões com ícones e animações hover  
✅ Layout responsivo e organizado  

---

# 📊 Estrutura da planilha XLS

O sistema exporta automaticamente os dados na seguinte ordem:

- NF
- CLIENTE
- TRANSPORTADORA
- TIPO
- DATA
- AÇÃO
- MOTIVO
